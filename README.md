# Atlas

Atlas is an AI-powered Career Operating System that helps discover, rank,
tailor and manage job applications.

## Vision

Instead of blindly applying to hundreds of jobs,
Atlas intelligently finds the opportunities that maximize interview probability.

## Current Version

v0.1.0

## Roadmap

- Candidate Profile
- Resume Engine
- Job Discovery
- Ranking Engine
- Application Tracker
- Browser Automation
- Interview Assistant
